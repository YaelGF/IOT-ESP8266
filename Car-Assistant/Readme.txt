 
Proyecto de carro de dos motores, un sensor Ultrasonico para evitar colicion, que se controla mediante google assistant

pines a usar:

motor izquierda:

en1 pin 4 (D2)
out1 pin 0 (D3)
out2 pin 2 (D4)

motor derecha:

en2 pin 13 (D7)
out3 pin 14 (D5)
out4 pin 12 (D6)

Sensor Ultrasonico:

echo 5 (D1) 
tigger 16 (D0)
